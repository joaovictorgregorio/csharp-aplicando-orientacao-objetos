## 📝 About This Repository

This repository contains the practical code developed during the course. Its main goal is to consolidate the fundamental concepts of **Object-Oriented Programming** applied to C#, such as classes, objects, attributes, methods, properties, and encapsulation.

It also includes course challenges completed during the classes in the `Challenges/` folder. The internal structure of this folder is intentionally not detailed here.

## 💻 How to Run the Code

Since this project was built using the **.NET CLI**, you can easily run it from your terminal.

1. Ensure you have the [.NET SDK](https://dotnet.microsoft.com/download) installed.
2. Clone this repository and navigate to the desired project folder.
3. Run the following command in the terminal:

```bash
dotnet run
```

## 📂 Repository Structure

| Path                           | Description                                               |
| ------------------------------ | --------------------------------------------------------- |
| ScreenSound/Program.cs         | Main entry point for the ScreenSound project.             |
| ScreenSound/ScreenFormatter.cs | Utility class for screen/header formatting.               |
| ScreenSound/Music.cs           | `Music` class with properties and display behavior.       |
| ScreenSound/Album.cs           | `Album` class that groups music tracks.                   |
| ScreenSound/Band.cs            | `Band` class representing artists/bands.                  |
| ScreenSound/Podcast.cs         | `Podcast` class for podcast data and episodes.            |
| ScreenSound/Episode.cs         | `Episode` class used by podcast entities.                 |
| ScreenSound/Genre.cs           | `Genre` enum/type used to classify music content.         |
| ScreenSound/ScreenSound.csproj | Project file for ScreenSound.                             |
| ScreenSound/ScreenSound.sln    | Solution file for opening and managing the project in IDE |
| ScreenSound/bin/               | Compiled output (generated at build/run time).            |
| ScreenSound/obj/               | Intermediate build artifacts (generated by .NET SDK).     |
